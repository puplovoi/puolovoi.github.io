23级Crypto培养方案

编程语言学习

  1. C语言知识巩固（3天）

  2. Python（20天）

PS：

  1. 特此说明，编程语言学习不特设时间，跟随后续学习内容共同进行

  2. 理论学习与实践做题也是同步的

理论学习

第一阶段——密码学数学基础

一. 数论（一周）

  1. 了解欧几里得算法及欧几里得扩展算法【1】

  2. 了解模运算【1】

  3. 了解素数、素性测试【1】

  4. 了解欧拉定理、费马小定理【1】

  5. 了解中国剩余定理【1】

  6. 了解离散对数【1】

二. 群论（一周）

  1. 了解并掌握群的概念及性质【1】

  2. 了解并掌握环的概念及性质【1】

  3. 了解并掌握域的概念及性质【1】

  4. 了解并掌握有限域（GF）的概念及性质【1】

  5. 了解并掌握多项式运算【1】

第二阶段——密码学体系学习

一. 古典密码体制（3天）

  1. 了解密码学的基本术语，如明文、密文、密钥等

  2. 了解并掌握凯撒密码及攻击方式【1】

  3. 了解并掌握维吉尼亚密码及攻击方式【1】

  4. 了解并掌握栅栏密码及攻击方式

  5. 自行查找并了解其他古典密码加密形式【1】

二.现代密码体制(42+10+3+12=67天)

（一）非对称密码(14+14+14=42天)

  1. 了解并掌握RSA的原理、掌握对RSA的常见攻击方式【14】

  2. 了解ECC的原理并掌握椭圆曲线加密的原理【14】

  3. 了解EIgmal的原理【14】

（二） 对称密码(1+3+3+3=10天)

  1. 了解分组密码、流密码的概念【1】

  2. 了解并掌握Feistel结构【3】

  3. 了解并掌握DES、三重DES的原理【3】

  4. 了解并掌握AES原理【3】

（三） 分组密码结构(3天)

  1. 了解并掌握5种分组密码的工作模式及优缺点【3天】

（四）杂凑函数(2+3+7=12天)

  1. 了解杂凑函数的概念、性质及基本结构【2】
  2. 了解并掌握MD5的原理及攻击方式【3】
  3. 了解并掌握SHA系列加密的原理【7】

实践做题

第一阶段——密码学数学基础

一. 数论（一周）

  1. 每天手写2道欧几里得算法题【2*5=10道】

  2. 熟练掌握对于数的模运算，每天自行练习【不限】

  3. 学习素性测试Python代码，读懂并且能够编写【5天】

  4. 每天手写3道欧拉定理运算【3*3=9道】

  5. 每天手写2道中国剩余定理题【2*2=4道】

  6. 深刻理解离散对数，概念深究【1整天】

  7. 进行练习的上交

二.群论（一周）

  1. 从群论（群、环、域）的理解角度入手，理解高等代数（线性代数也行），并写出学习报告【3天】

  2. 对于抽象代数系统进行归纳，撰写学习报告【1天】

  3. 每天求解2道有限域（GF）的多项式运算【1*2=2道】

  4. 进行学习报告的上交

第二阶段——密码学体系学习

一. 古典密码体制(3天)

  1. 尝试应用已经学习的数学理论解释各个古典密码体制的数学原理【3天】

二.现代密码体制

（一）非对称密码(14+14+14=42天)

  1. 了解并掌握RSA的原理、掌握对RSA的常见攻击方式

a.每周4道RSA的题（先在buuctf上做）

b.形成WP题解，并间接地学习Python代码（14天）

c.每周上交WP题解

d.每周进行WP题解的上交

PS：

此后可以开始每周做题（节假日除外），保证每周4道题动态平衡（题目具体知识点不限）

学习其他理论的过程，具体日期也可以灵活分布，但是RSA是基础，所以RSA及之前的顺序要严格遵循

第三阶段——以赛促练(学完RSA就可以开始了)

每周参加各种比赛

做题&比赛平台：buuctf，nssctf 这两个足够用了

考核标准

一.假期学习【100分 = 理论学习学习报告50%+实际做题50%】

二.返校考核【100分 = 笔试60% + 面试40%】
